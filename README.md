# GSM_Dust-data
> 광주SW마이스터고의 미세먼지 정보를 실시간으로 제공하는 프로젝트

## 프로젝트 내용
정부에서 제공하는 미세먼지 데이터는 실시간이 아니고, 모든 구역에 측정기가 배치되지 않아 부정확한 데이터를 전달받을 확률이 높다. 이에 우리는 미세먼지 데이터의 신뢰성을 높이기 위해 미세먼지 데이터를 실시간으로 측정 해 데이터를 제공하는 교내 미세먼지 정보 제공 API 시스템을 개발하게 되었다.
